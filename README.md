# 3010WBORUXM

### Install Guide:

- Wipe dalvik, cache, metadata,
- Flash ROM.zip,
- Wipe dalvik, cache, metadata,
- Format data,
- Reboot in fastboot,
- Execute fastboot erase frp,
- Execute fastboot -w,
- Reboot,
- Restart your device after completing the initial setup,

### Rom features
- Pure Global
- Kigurumi depth wallpaper
- Fixed BT playback in browser
- Debloat
- AOD Removed
- Give rom 30 hours to settle
- E404 kernel (KSU1.0.5)
